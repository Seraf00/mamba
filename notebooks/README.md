# Notebooks Directory

This directory contains Jupyter notebooks for data exploration, model comparison, and explainability demonstrations.

## Available Notebooks

### 1. `data_exploration.ipynb`
- CAMUS dataset visualization
- Image quality analysis
- Label distribution analysis
- Ejection fraction distribution

### 2. `model_comparison.ipynb`
- Compare base vs Mamba-enhanced models
- Visualize performance metrics
- Statistical significance tests
- Parameter efficiency analysis

### 3. `explainability_demo.ipynb`
- Grad-CAM visualizations
- Attention map analysis
- Mamba state dynamics
- Uncertainty estimation
- Clinical report generation

## Usage

```bash
jupyter lab notebooks/
```

Or open directly in VS Code with Jupyter extension.
